# Changelog

All notable changes to the Easing Wizard Claude plugin will be documented in this file.

## 1.0.0 - 2026-04-02

### Added

- 4 skills: recommend-curve, improve-easings, audit-easings, easing-advice
- Animation expert agent for motion design consultation
- MCP integration with 7 curve generation tools (Bezier, Spring, Bounce, Wiggle, Overshoot)
- Session start hook for auto-detecting animation projects
- 68 curated presets across 5 curve types
- CSS and Tailwind output support
- Preview link generation for all curves
